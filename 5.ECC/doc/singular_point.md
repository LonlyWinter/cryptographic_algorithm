# 奇点（Singular Point）

### 一、什么是“奇点”（Singular Point）？

在代数几何中，**奇点**是指曲线上某些“不光滑”或“异常”的点，比如：

- **尖点**（cusp）：曲线在此处突然折返，形成尖角；
- **自交点**（node）：曲线在此处交叉自身。

这些点会导致曲线在该处**不可导**或**没有良好定义的切线**，从而破坏群结构所需的代数性质。

### 二、为什么椭圆曲线不能有奇点？

椭圆曲线密码学依赖于**椭圆曲线上点构成的阿贝尔群**（Abelian group），其中最关键的操作是**点加法**（point addition）。这个加法运算是通过几何方法（如作切线、连线等）定义的，而这些操作**要求曲线处处光滑**——即**没有奇点**。

如果曲线存在奇点：
- 点加法可能无法定义；
- 群的封闭性、结合律等性质可能失效；
- 密码系统的基础（离散对数问题）将不再成立，安全性崩溃。

因此，**必须排除有奇点的曲线**。

### 三、如何判断一条椭圆曲线是否有奇点？

对于定义在域（如实数域或有限域 $\mathbb{F}_p$）上的短 Weierstrass 形式椭圆曲线：

$$
y^2 = x^3 + ax + b
$$

它**无奇点的充要条件**是其[**判别式**（discriminant）](./discriminant.md)**不为零**：

$$
\Delta = -16(4a^3 + 27b^2) \neq 0
$$

因为在密码学中通常工作在特征 ≠ 2, 3 的有限域（如大素数域 $\mathbb{F}_p, p > 3$），常数因子 -16 可忽略（非零），所以简化为：

$$
\boxed{4a^3 + 27b^2 \not\equiv 0 \pmod{p}}
$$

> 满足这个条件 → 曲线光滑 → 可用于 ECC  
> 不满足条件（等于 0） → 曲线有奇点 → **不能用于密码学**

### 四、直观例子（在实数平面上）

1. **正常曲线**（无奇点）：  
   $ y^2 = x^3 - x $  
   → $ a = -1, b = 0 $  
   → $ 4(-1)^3 + 27(0)^2 = -4 \neq 0 $ → 光滑

2. **有奇点的曲线**（例如尖点）：  
   $ y^2 = x^3 $  
   → $ a = 0, b = 0 $  
   → $ 4(0)^3 + 27(0)^2 = 0 $ → **有奇点（在原点）**，不可用！

   这条曲线在 (0,0) 处是一个尖点，无法定义合理的点加法。

### 五、总结

> “确保曲线无奇点” 就是为了保证椭圆曲线是**光滑的代数曲线**，从而使其上的点能构成一个**数学结构良好、运算封闭的群**，这是 ECC 安全性和正确性的基础。

在实际 ECC 标准（如 NIST、SECG）中，所有推荐的曲线参数都经过严格验证，确保 $4a^3 + 27b^2 \not\equiv 0 \pmod{p}$，即**绝对无奇点**。
